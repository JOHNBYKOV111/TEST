Данный скрипт "TEST" позволяет анализировать данные о производительности сотрудников из CSV-файлов и формировать отчет,
показывающий среднюю производительность по каждой должности.

Для корректной работы проекта необходимы следующие пакеты Python:

pip install csv tabulate argparse pytest

Создание и запуск виртуального окружения:

python -m venv venv - создание виртуального окружения.

venv\Scripts\activate - запуск виртуального окружения.

Запуск программы:

python main.py --files employees1.csv employees2.csv --report report.txt

Где:

--files - аргумент, указывающий входные CSV-файлы.

--report - аргумент, указывающий выходной файл отчёта. Если не указан, отчёт выводится в консоль.

Вызов справки:

python main.py --help

Запуск тестов:

pytest main.py
